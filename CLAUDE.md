## Project Overview

Soteria Legal is a static website hosting legal documents (privacy policy, terms of service) for the Soteria application.

## Structure

- `index.html` - Main landing page
- `privacy.html` - Privacy policy
- `terms.html` - Terms of service

## Development

No build tools required. Files are static HTML and can be opened directly in a browser or served with any static file server.
